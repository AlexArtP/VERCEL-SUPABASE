El formulario de registro contiene todos los campos:
- Nombre, Apellido Paterno, Apellido Materno
- RUN con validación
- Profesión (dropdown)
- Sobre ti, Cargo actual, Email, Teléfono
- Contraseña con validación

Archivos principales:
- components/RegisterForm.tsx (Formulario principal)
- components/RegistrationModal.tsx (Modal wrapper)
- app/api/auth/register/route.ts (API endpoint)
